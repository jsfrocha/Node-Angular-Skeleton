Skeleton for AngularJS + NodeJS Application
===========================================

Requirements
------------

- Node.JS & NPM
- Bower
- Gulp
- Compass

Contents
--------

- Express.JS
- Angular.JS
- Angular.JS UI-Router

Quick Start
-----------

Install Required Node Modules
> npm install

Install Required Bower Components
> bower install

Start the Application on `localhost:3000`:
> npm start
